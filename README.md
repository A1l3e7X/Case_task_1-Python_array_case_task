#Кейс-задача: сумма отрицательных элементов массива между максимальным и минимальным
## Описание задачи

Дан одномерный массив А размерности N. 
Найти сумму отрицательных элементов, расположенных между максимальным и минимальным. 

## Цель работы

Разработать программу на языке **Python**, которая:
- принимает одномерный массив;
- находит минимальный и максимальный элементы;
- определяет их индексы;
- вычисляет сумму отрицательных элементов между ними;
- выполняет проверку корректности вводимых данных;
- выводит подробные результаты работы.

## Используемый язык программирования

- **Python 3.14.4**

## Алгоритм решения

1. Запросить у пользователя количество элементов массива.
2. Проверить корректность введённого значения.
3. Если размер массива меньше или равен нулю, завершить выполнение программы.
4. Ввести элементы массива.
5. Найти минимальный элемент массива.
6. Найти максимальный элемент массива.
7. Определить индексы минимального и максимального элементов.
8. Найти границы участка между этими индексами.
9. Выбрать отрицательные элементы, расположенные между минимальным и максимальным элементами.
10. Вычислить их сумму.
11. Вывести подробные результаты работы программы.

## Код программы

```python
def get_int_input(prompt):
    """Безопасный ввод целого числа."""
    while True:
        try:
            return int(input(prompt))
        except ValueError:
            print("Ошибка: введите целое число.")


def main():
    # Ввод количества элементов массива
    n = get_int_input("Введите количество элементов массива: ")

    # Проверка размера массива
    if n <= 0:
        print("Массив пуст или его размер задан некорректно.")
        return

    # Ввод элементов массива
    a = [get_int_input(f"Введите элемент A[{i}]: ") for i in range(n)]

    # Нахождение минимального и максимального элементов
    min_value = min(a)
    max_value = max(a)

    # Нахождение индексов первого вхождения минимального и максимального элементов
    min_index = a.index(min_value)
    max_index = a.index(max_value)

    # Определение диапазона между минимальным и максимальным элементами
    start, end = sorted([min_index, max_index])

    # Вычисление суммы отрицательных элементов между ними
    negative_sum = sum(x for x in a[start + 1:end] if x < 0)

    # Вывод результата
    print("\nИсходный массив:", a)
    print("Минимальный элемент:", min_value)
    print("Максимальный элемент:", max_value)
    print("Индекс минимального элемента:", min_index)
    print("Индекс максимального элемента:", max_index)
    print("Сумма отрицательных элементов между минимальным и максимальным:", negative_sum)


if __name__ == "__main__":
    main()

#Проверка if __name__ == "__main__": гарантирует, что точка входа в программу (в данном случае функция main()) сработает только при прямом запуске файла.
Это позволяет безопасно импортировать отдельные функции из этого файла (например, get_int_input) в другие.

##Пример работы программы

##Пояснение к работе программы
Программа сначала запрашивает количество элементов массива и проверяет корректность введённого значения. Если пользователь вводит не целое число, программа выводит сообщение об ошибке и предлагает повторить ввод.
После этого вводятся элементы массива. Затем программа находит минимальный и максимальный элементы, определяет их индексы и выделяет участок массива, расположенный между ними. Из этого участка выбираются только отрицательные элементы, после чего вычисляется их сумма.
В завершение программа выводит подробную информацию о результатах работы: исходный массив, минимальный и максимальный элементы, их индексы, а также сумму отрицательных элементов между ними.
